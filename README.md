***Bank Account Management System (Python Project)***
    **Description**
         This is a Python-based mini banking system built using OOP (Object-Oriented Programming) concepts.
  It allows users to create accounts, log in securely, deposit and withdraw money, check balances, and view all existing accounts.

**Features**
  Create a new bank account with a unique account number
  Secure login using password (hidden input using getpass)
  Deposit and withdraw money
  View current balance
  Display all existing accounts (for admin/testing)
  Remove an account
  Random account number generation
  
**Concepts Used**
  Object-Oriented Programming (class, __init__, methods)
  Conditional statements (if-else)
  Loops
  Functions and user input handling
  Python modules: random, getpass
      
**Technologies**
   Language: Python
   IDE: Jupyter Notebook / VS Code
**How to Run**
  Open this project in Jupyter Notebook or any Python IDE.
  Run the program.
  Choose options from the menu to:
  Create a new account
  Login to an existing account
  Deposit / Withdraw / View balance
